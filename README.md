# RAG-based-course-companion
This repository contains a RAG based system that would help students to fetch specific videos and timestamps of the topic they are looking for.

Add your course videos inside a folder named videos and run the files in the following order :
1. processed.py
2. speech_json.py
3. embeddings.py
4. preprocess_json.py
5. app.py

Required :
Ollama Installed on your system (ollama run llama3.2 - run this command in terminal before executing 4.)
